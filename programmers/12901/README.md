## 풀이

7로 나눈 나머지값으로 요일을 정한다

## 후기

new Date의 getDate메소드를 알고 있다면 금방 풀겠지만 모른다면.. 어떻게 해야할까
처음에는 각 월별로 며칠인지 외우고 있어야 풀 수 있는건가..? 싶었다
