## 풀이

split으로 하나씩 나눈 후 
reduce로 순회하면서 해당 요소가 * / # / 그 외 문자 / 숫자
4가지 경우에 따라 달리 동작하도록 했다.

'*' : 현재까지 축적된 acc의 앞, 앞앞 요소(가 있다면)를 2배로 

'#' :이면 앞 요소에 -1을 곱하고

'S, D, T' : dart 리터럴에 넣어 나오는 수만큼 앞 요소 제곱

number : 축적값 마지막이 숫자면 문자열로 합치고 다시 숫자, 아닌 경우에는 그냥 acc에 추가

그 후에 모든 요소를 다 더한다.
