## 풀이

isWriteStr(p) - 입력된 문자열이 올바른 괄호 문자열인지 판별

getDiviseIndex(p) - 입력받은 p에서 더 이상 나눌 수 없는 균형잡힌 문자열 u와 나머지v를 가르는 index값을 찾아 반환

divise(p) - 입력받은 p를 u, v로 나누어 배열로 반환 [u, v]

convert(u, v) - u가 올바른 괄호 문자열이 아닐 때 u와 v를 적절히 처리해서 반환
